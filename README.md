# funx-theme
> element component funx theme.


## Installation
```shell
npm i funx-theme -S
```

## Usage

Use Sass import
```css
@import 'funx-theme';
```

Or Use webpack
```javascript
import 'funx-theme';
```

Or
```html
<link rel="stylesheet" href="path/to/node_modules/funx-theme/lib/index.css">
```

##  Import on demand
```javascript
import 'funx-theme/lib/input.css';
import 'funx-theme/lib/select.css';

// ...
```
